# Detect-MUBs-by-ChatGPT
Benchmarks, prompts and results for detecting MUBs by ChatGPT


## Overview

contracts folder contains the contracts evaluated in the work, which is forked from [Web3Bugs](https://github.com/ZhangZhuoSJTU/Web3Bugs/tree/main/contracts).

reports folder contains the original code4rena reports and the output by ChatGPT (general prompts and guidance prompts).

results folder contains the [prompts file](results/prompts), the [bug category](results/bugCat) and the analysis results in form of excels, which are extracted from the folder of reports.
