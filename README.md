# Detect-MUBs-by-ChatGPT
Benchmarks, prompts and results for detecting MUBs by ChatGPT

prompts file contains the prompts prefixed to the file to ChatGPT for each kind of bugs.
results folder contains the analysis results in form of excels, which are extracted from the folder of reports.
reports folder contains the original code4rena reports and the output by ChatGPT (general prompts and specific prompts).
contracts folder contains the contracts evaluated in the work, which is forked from (https://github.com/ZhangZhuoSJTU/Web3Bugs/tree/main/contracts).
